A projektem egy fiktív autókereskedésnek a termékkatalógusát valósítaná mega vásárlók számára a 4 kapott tétel felhasználásával.
Magukat az autókat egy saját XML fájlból importálom az alkalmazásba(1.tétel). 
A felhasználói felület egyszerűnek mondható, minden egyes autó objektumhoz létrehoz egy hozzá kapcsolódó gombot(ez a gombokból álló UI tétel).
A gombok megnyomására a fő területen megjelnnek az aktuális autóhoz kapcsolódó egyéb információk és adatok(évjárat, ár stb.)
Az adatokat egy terméklistához hasonlóan ki lehet írni gombonyásra egy csv fájlba, ez is egy tétel.
